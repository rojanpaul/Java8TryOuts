Lambda expressions are only Implementation function of Functional Interface .
Variables defined inside are local variables of functions.
this.varibaleName refers to instance variable of the class where lambda expression is defined.

Annoymous inner classes are Class implementation of Interface/Solid Classes. Anonymous inner classes are more powerful.
Varibales defined inside are instance variables.
this.varibaleName refers to instance variable of the anonymous class, NOT the class in which it is defined..
